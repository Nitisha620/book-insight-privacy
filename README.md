# Privacy Policy for Book Insight

Book Insight is a Chrome extension that helps users quickly view information about books from any webpage.

## Information Collection

Book Insight does not collect, store, or share any personal information from users.

## How the Extension Works

When a user selects text on a webpage and activates the extension through the right-click menu, the selected text is used to query the Google Books API in order to retrieve book information such as title, author, description, and ratings.

## Data Usage

The extension only sends the selected text to the Google Books API to retrieve book details. No personal or browsing data is stored, tracked, or shared.

## Third-Party Services

Book Insight uses the Google Books API to retrieve book information. The extension does not control how Google processes requests sent to its API.

## Changes

This privacy policy may be updated if the extension’s functionality changes.

## Contact

If you have questions about this privacy policy, please contact the developer through the Chrome Web Store listing.
